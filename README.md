# 📝 Scrum Product Backlog - User Stories

**Jogabilidade Base (Batalha Naval)**
* US01: Como jogador, quero posicionar os meus navios de forma manual ou automática (aleatória) no tabuleiro, para poder iniciar a partida rapidamente.
* US02: Como jogador, quero selecionar células no tabuleiro do adversário para efetuar disparos e tentar afundar a frota inimiga.
* US03: Como novo jogador, quero aceder à secção de guias de jogo (Game Guides), para compreender as regras específicas e mecânicas desta versão online.

**Interação Social e Comunidade**
* US04: Como utilizador, quero adicionar outros jogadores à minha lista de amigos, para ver quando estão online e desafiá-los facilmente.
* US05: Como jogador, quero utilizar um chat integrado (Global ou In-game), para comunicar e interagir com o meu adversário ou com a comunidade durante as partidas.
* US06: Como jogador competitivo, quero consultar o meu histórico de partidas (Match History), para analisar as minhas vitórias, derrotas e estatísticas gerais na Batalha Naval.

**Personalização e Economia Virtual (Loja)**
* US07: Como utilizador, quero aceder à loja virtual, para adquirir moedas virtuais (Virtual Coins) do jogo.
* US08: Como utilizador, quero usar as minhas moedas virtuais para comprar avatares exclusivos e pacotes de emojis, para personalizar a minha identidade e expressões durante o jogo.
* US09: Como fã da plataforma, quero ter acesso a uma loja de merchandising externa, para poder comprar artigos físicos da marca Papergames.

**Torneios e Competição**
* US10: Como organizador ou jogador experiente, quero ter a capacidade de criar um torneio personalizado (Create Tournament), definindo regras e número de participantes.
* US11: Como jogador competitivo, quero um painel de gestão (My Tournaments), para acompanhar as chaves, os resultados e o meu progresso nas competições em que estou inscrito.

**Subscrições e Acessibilidade (Plataforma)**
* US12: Como utilizador assíduo, quero poder subscrever um plano Premium (Plus ou Gruppo), para usufruir de uma experiência sem anúncios e desbloquear funcionalidades exclusivas de personalização.
* US13: Como utilizador de dispositivos móveis, quero poder descarregar a aplicação nativa para iOS ou Android, para jogar Batalha Naval com uma interface otimizada para o meu telemóvel.
* US14: Como programador terceiro, quero aceder à documentação e portal para Developers, para poder integrar funcionalidades da plataforma ou criar serviços associados (Games as a Service).
